# Чат-Бот для групп ВК
Бот в основном предназначен для ответов для простых вопросов

## Подготовка
* Установка пакетов через pip
  * $ pip install pymongo
  * $ pip install vk_api
* Добавление настроекв файл propirties.py
  * token
  * groupID
  * adminsId
## Используемые пакеты
* re
* time
* vk_api
* pymongo
